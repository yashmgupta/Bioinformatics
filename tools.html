Bioinformatics Tools & Projects

Explore a collection of tools and projects developed for DNA sequence analysis and bioinformatics research.

DNAConservation

Conservation & Entropy Analyzer: Upload or paste FASTA sequences, align, and visualize nucleotide/k-mer conservation and entropy.

Key Features: FASTA input, alignment, conservation & entropy visualization.

Relevance: Comparative genomics, identifying functional regions.

View on GitHub

GeneBank-Genie

A versatile tool for the analysis of GenBank records.

Key Features: GenBank file parsing, data extraction, sequence manipulation.

Relevance: Facilitates working with GenBank database.

View on GitHub

Microsatellites CNN-RNN

Efficient Classification of DNA Microsatellites Using a Hybrid CNN-RNN Model.

Key Features: Deep learning, microsatellite classification.

Relevance: ML for identifying repetitive DNA elements.

View on GitHub

Representing-DNA

Representing DNA for Machine Learning Algorithms: A Primer on One-Hot, Binary, and Integer Encodings.

Key Features: Encoding methods explained, examples, educational.

Relevance: Foundational for applying ML to DNA sequences.

View on GitHub

Nucleotide Analysis

Jupyter notebook of alternative functions for nucleotide string analysis and plotting.

Key Features: Sequence statistics, GC content, plotting utilities.

Relevance: Practical examples for basic DNA analysis.

View on GitHub

DNA Barcode Classification

A quantum-inspired tensor network-based machine learning model to categorize cytochrome c oxidase subunit I (COI).

Key Features: Tensor network model, COI classification.

Relevance: Advanced ML for DNA barcode-based species ID.
